Лабораторна робота 5

CI/CD розгортання в хмарі

Вебзастосунок автоматично збирається через GitHub Actions, публікується в Docker Hub і оновлюється на Azure VM за допомогою Watchtower.

01

Commit

Зміни відправляються у гілку main GitHub-репозиторію.

02

Lint

Pipeline перевіряє форматування HTML і CSS через Prettier.

03

Build & Push

Docker image збирається і завантажується в Docker Hub.

04

Deploy

Watchtower на VM отримує новий image і перезапускає контейнер.
